## 📋 推荐列表格式

在 `推荐列表.ini` 文件中添加自定义软件，格式如下：

```
-------------在下面添加--------------------
["QQ",3,"腾讯"],
["360安全卫士",1,"360"],
["微信",1,"腾讯"],
["WPS",1,"腾讯"],
--------------在上面添加-------------------
```

**格式说明：**
1. 格式为 `["搜索词","搜索到的第几行","软件库"]`
2. 软件库只能填 **腾讯** 和 **360**
3. 建议添加前先搜索一次找到准确位置
4. 请在分割线内添加，其他地方添加无效
