# 推荐列表格式 
          1、格式为每行["搜索词","搜索到的第几行","软件库"], 
          2、软件库只能填腾讯和360，不能填其他。
          3、建议添加前搜索一次找到位置。
          4、请在分割线内添加，其他地方添加无效。
          -------------在下面添加--------------------
          ["QQ",3,"腾讯"],
          ["360安全卫士",1,"360"],
          ["微信",1,"腾讯"],
          ["WPS",1,"腾讯"],
